# 100倍币的逻辑真相
## 百倍，一定是非共识。

### 1.投赛道，不投项目。第1,2,3全投，而不是只投第一名。
如2020的dex，uni,shushi都投，结果都赚。

### 2.一定要寻找未共识的
非共识阶段，市值非常小。再到共识。
如果你看到一个币，大家都觉得很好，很难在一个周期100倍的。

Meme百倍币的三级火箭：链上、小所、大所。例如狗狗，shibi。流量再提决定几个零。

交易所很难找到，诞生在链上。如shib，2020.8诞生在链上。
